HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element from the head would be displayed in the tab of web page.
  • There should be only one h1 element in a page.
  • The header numbers should be used in order, h1 through h6.
  • The elements should be semantic for more readability.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Period sign means class in CSS
  • We can use semantic text or hexadecimal code for colors
  • According to CSS rule, selector is followed by declaration block
  • CSS elements have properties and respectives values for those properties
  • Each CSS selector is an attribute

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls any updates that are available in the main branch
  • git commit: commits the changes
  • git push: pushes the commited changes to main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is OOP language
  • Function has declaration block or a body where code is executed
  • Besides getting declared, function needs to be called in order to be used
